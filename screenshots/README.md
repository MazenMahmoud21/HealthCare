This folder contains screenshots of the Healthcare Management System.

Use the following naming convention for consistency:

- `01-login.png` - Login page
- `02-register.png` - Registration page

### Admin Dashboard
- `03-admin-dashboard.png` - Admin dashboard overview

### Doctor Dashboard

### Patient Dashboard

### Appointments

## How to Take Screenshots

1. Open the application in browser: `http://localhost:5143`
2. Navigate to each page listed above
3. Save with the corresponding filename
4. Ensure images are PNG format for best quality

## Image Optimization Tips

- Use PNG format for lossless quality
- Keep file size under 1MB for each screenshot
- Capture the full page or relevant content area
- Ensure text is clearly readable
